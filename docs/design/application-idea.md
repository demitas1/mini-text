# mini-text アプリケーションのアイディアメモ

docs/examples/typeinto.sh
docs/examples/copyfrom.sh

これらはInkscapeのテキストボックスに任意の文字列を入力する機能と、逆にテキストボックス中の文字列をコピーする機能の実装コンセプトを示したものです。

これを元に、以下のようなアプリケーションを作成したい

- Linux, X11デスクトップで動作
- xdotool, xclip がインストールされている前提
- pyqt6 を用いてデスクトップ上に小さなウィンドウを表示する
- デスクトップ上にあるウィンドウ一覧を表示するリストボックス
- リストボックスから目的のアプリケーションウィンドウを選択
- テキストボックスに入力した文字列をアプリケーションに送信 (typeinto)
- アプリケーションから文字列を選択コピーしテキストボックスにコピーする (copyfrom)
- 送信ボタン
- コピーボタン
- mini-textアプリケーションは通常最前面に表示
